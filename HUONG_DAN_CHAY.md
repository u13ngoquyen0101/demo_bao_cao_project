# Hướng Dẫn Chạy Ứng Dụng Web

## Yêu cầu
- Python 3 đã cài đặt

## Các bước

### 1. Mở Terminal
- Windows: nhấn `Win + R`, gõ `cmd` (hoặc `powershell`), Enter
- Hoặc mở VS Code → Terminal → New Terminal (`Ctrl + `` `)

### 2. Vào thư mục dự án
```bash
cd C:\Users\TunDat\Desktop\demo_bao_cao_project
```

### 3. Chạy server Flask
```bash
python app.py
```

### 4. Mở trình duyệt
- Mở Chrome / Edge / Cốc Cốc...
- Gõ vào thanh địa chỉ: **http://127.0.0.1:5000**
- Nhấn Enter

### 5. Tắt server
- Quay lại terminal
- Nhấn `Ctrl + C`
